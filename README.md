# 天工机巧盘 · 拼图推演

《诛仙世界》S4 天工机巧盘的纯静态拼图求解工具。玩家可以录入背包中五类机巧石的全部库存，工具会自动挑选当前盘面需要的若干块，并搜索恰好铺满棋盘的摆放方案。

## 使用

直接双击 `index.html`，无需安装依赖、无需联网。

1. 选择已经开放的机巧盘等级。
2. 录入背包中的方石、左弯石、丁字石、长条石和右弯石数量。
3. 库存总数达到当前盘面最低需求后，点击“开始推演”。
4. 工具会自动选石；盘面和结果提示会列出本方案实际使用的形状与数量。
5. 在“落石次序”中查看旋转方向与具体格位。

库存可以多于盘面所需数量，每种形状最多可录入 99 块。“填入示例”会按当前等级填入一组已经验证可解的库存；“清空”可以一键将五类库存全部归零。

## 当前规则

- 每块机巧石占 4 格。
- 机巧石允许旋转，不允许镜像。
- 左弯石和右弯石是两类不同库存。
- 自动选择的机巧石不得超过背包中对应形状的数量。
- 所有盘面格位必须恰好铺满，不允许重叠、越界或留空。
- 最多展示前 12 个可行方案。

## 棋盘数据说明

完整棋盘外框共 40 格：顶部居中的 4 格，加上下方 6 列 × 6 行的 36 格。

1～4 级开放区域已依据游戏实图确认，分别为 16、20、24、28 格。5～7 级已知会依次增加到 32、36、40 格，但具体新增位置尚未确认，因此目前在界面中显示为锁定状态，不能选择或参与推演。待获得对应等级实图后，再在 `solver-core.js` 的 `BOARD_LEVELS` 中补充准确格位。

## 验证

安装 Node.js 后可运行：

```powershell
npm.cmd test
```

测试覆盖棋子旋转、库存不足、库存富余自动选石、1～4 级基准铺法、四级实图格位、40 格完整外框、5～7 级锁定状态、精确覆盖和无解组合。

重新生成单文件分享版：

```powershell
npm.cmd run build:single
```
